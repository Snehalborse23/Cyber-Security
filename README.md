The Caesar Cipher is one of the simplest encryption techniques.  
Each letter in the plaintext is shifted by a fixed number of positions in the alphabet.  
- **Encryption:** shifts letters forward.  
- **Decryption:** shifts letters backward by the same shift value.

Encrypted : KHOOR
